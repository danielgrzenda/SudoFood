API Reference¶

yummly_api.build_list()[source]¶

For every 10 recipes pulled, this function dumps said recipes into a single json file and uploads it to the S3 bucket.

yummly_api.get_one_recipe(recipe_id)[source]¶

Returns the corresponding recipe for each recipe id

recipe_id : str
Recipe id that was pulled from search recipe
yummly_api.recipe_search(search_params=None)[source]¶

Returns recipe ids from YummlyAPI based on the given search parameters.

search_params : dict
Any parameters that the user wishes to search by
